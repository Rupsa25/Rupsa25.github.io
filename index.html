Intro

I recently graduated with a Master's Degree in Computer Science from Rutgers University.I completed my Bachelors Degree in Computer Science and Engineering from Vellore Institute of Technology,Chennai.I have 6 months of experience working as a Data Engineer at Phenom People, an HR tech firm.

Academic Projects

  • SQL Query Executer

    Built a multi database query parsing engine that executes SQL queries in different databases namely MySQL and Redshift.Reduced the response time by 35% with the help of indexing and built this as a cross platform mobile application for android and iOS using react native for frontend and flask backend

  • KOBE

    Developed a web application is designed to provide in-depth analysis of the NBA leagues for team managers, analysts and betting enthusiasts. It aims at combining data visualisation techniques with data mining to extract information (patterns) that can be utilized during auction season or while planning match lineups/strategies.

  • CampR

    Built CampR, a user friendly web application which allows users to search and view campgrounds with ease!It allows for people to create posts and reviews regarding campgrounds.Built using MongoDB, Express and Node for seamless handling of routing, views, controllers, and database operations,it is deployed over heroku.

  • Massive Data Mining

    Implemented a MapReduce program using PySpark that implements a simple “People You Might Know” social network friendship recommendation algorithm.Market Basket Analysis (MBA) a technique used by retailers to understand the purchase behavior of their customersis used to write an algorithm to find products which are frequently browsed together.

  • What's Cooking

    Developed a model to predict the type of cuisine from the list of ingredients given using support vectors classifier. Incorporated the concepts of text mining for the initial pre-processing such as use of lemmatization,Tf-idf on the text data and achieved a final test score accuracy of 81.043% with the highest ranked on leaderboard score being 83.216%.

  • Formula 1 Project

    Analyzed Formula 1 racing data and created dashboards by first ingesting and transforming data using Azure Databricks. Leveraged Delta Lakehouse features to perform operations on data using PySpark and SQL. Used Azure Data Factory to schedule the pipeline and incorporated an incremental load architecture for data ingestion using the merge operations on data stored in delta format.

  • Netflix Tableau Viz

    Created an interactive visualization using tableau for the netflix titles dataset.

  • Goodreads Tableau Viz

    Created an interactive visualization using tableau for the goodreads books dataset.
